Gathering

The Gathering page displays orders for today.

Each row of the table contains the following information:

  • Order Id - the auto-generated ID of the order

  • Customer Name - customer's firstname, lastname and e-mail

  • Amount of Food dishes - the number of portions of a dish, which the customer has selected on the Ordering page in the app

  • Pickup date and time - data and time of the pickup, which the customer has selected on the Ordering page in the app

  • Store - the pickup address, which the customer has selected on the Ordering page in the app

  • Deliver- status of the order. Click the link Issued / Utlevert to change status of the order

Orders By Day

The Orders By Day page displays the following information for the selected day:

  • Name of dish. The table is sorted by this field by default.

  • Amount - the number of portions, which the customer has selected on the Ordering page in the app

  • Pickup place- the pickup address, which the customer has selected on the Ordering page in the app

  • User data - customer's e-mail

Daily Dishes Matrix

The Daily Dishes Matrix displays the matrix of orders for the selected store and week by portions.

The first column Quantity displays all the prtion types available to select in the app (1, 1.5, 2, etc.).

Each day is presented by 2 columns - for the two dishes presented in the menu this day (Mandag 1, Fiskeboller & Mandag 2, Vegetarpølser med potetmos og stekte grønnsaker.)

The matrix displays the number of portions of each type for every dish ordered for particluar day.
