In order to manage dishes, navigate to Catalog > Products > Manage Products.

To add a new dish, click the Add new button.

To edit an existing dish, click Edit on the right to the needed dish.

The Product Edit page appears

Here you can edit product info and pictures, as well as add a dish to the daily menu in the app:

  • Add dish name

  • Add short description

  • Specify dish price

  • If you want the dish to appear in the menu for a specific date, please keep in mind that 2 dishes only can be added to the daily menu. Perform the following steps:

    • Click the Add new date button under the dish price.

    • Select a date from the drop-down list.

    • Specify the order of appearance in the daily menu (1 or 2). Optional step, you can skip it.

    • Click Update

  • When you are ready with edits, click Save or Save and continue if you are going to edit information on the other tabs.
