In order to edit / activate / deactivate the discount, navigate to Promotions > Discounts.

The MiddagsHuset app API is designed for a single discount — Discount for new users for maximum 5 first orders made during 24 hours since the first login. So, from the admin panel, it is possible to edit only the existing discount. If you add a new discount, it won't be visible in the app.

Click the Edit link in the table below:

The Edit Discount page appears:

Here you can do the following:

  • Edit Discount name

  • Set the discount amount in Percentage or in NOK. To set discount amount in percentage, select the Use Percentage checkbox and specify the amount in the Discount percentage field. You can also set a Maximum discount amount in NOK. Otherwise, specify the discount amount in NOK.

  • Specify Start date and End date - time range when the discount is active for new users.

  • When you are ready with edits, click Save

Now, the discount banners with the 24 hours timers will be visible in the app for the new users within the range specified in the Start date and End date fields.
