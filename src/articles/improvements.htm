Inspo

Added an information message on the "Inspo" page if the user has no fixlists with a category

Steps to reproduce:

  • Create a fixlist without categories

  • Tap the "Inspo" tab

Expected result: The user sees an informational message: "Please select a category to be able to see the information on this page" or something like this.

Actual result: The "Inspo" page is empty.

Expanding text under the photo

Precondition: User should have at least one inspo with a large number of characters (> 300 symbols) in the "Text" field (backoffice) on the "Inspo" page.

Steps to reproduce:

  • Log in to the account

  • Tap the "Inspo" tab

Expected result: Some of the text is hidden if the "Text" field contains a large number of characters, but all the text can be seen after clicking on it.

Actual result: Possible to see all text.

Additional info: Would be nice to hide some part of the name or text from the "Text" field in the backoffice on the "Inspo" page.

Dynamic Ads

Hardcoded Inspo became dynamic and manageable via admin panel.

  • Added respective pages to the backend website for the admins to be able to manage the sources of the Inspo tab content.

  • API and Image storage implemented

  • Made the app using/displaying custom content loaded dynamically from different sources of the advertisement elements


Fixlist

Implemented the Fixlist edit/update screen

Created a separate Fixlist screen for editing updating the Fixlist only data.

The screen displays the number of photos and participants, allows renaming the fixlist, and contains 3 tabs: Participants, Attachments, Categories


Chats

Removed chats for fixlists

Chats are now available only for photos.

Removed the "Open Chat" button.

The chats tab leads to the active fixlist photo chat.


Notifications

Added support for push notifications

3 types of notifications implemented:

  • IF there is a person accepting your Participation request
    (name) has accepted your invitation

  • IF there is a new chat from a photo or a Fixlist
    (name) has sent you a message

  • IF there is a new photo added to one of the Fixlists you are a participant of
    (name) has added a photo
