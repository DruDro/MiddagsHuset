In order to manage pick-up lcoations, navigate to System > Shipping > Shipping Methods.

The Shipping Methods page appears:

If you want to edit an existing pick-up address, click the Edit link next to the needed address:

If you want to add a new pick-up address, click the Add new button:

The Edit Shipping Method Details or Add a new Shipping Method page appears respectively. They are identical:

Here, just specify the new address in the Name field and click Save.

Now, the new address is updated or created in the database.
